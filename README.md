# Makarand Pundlik – Portfolio

Single-page portfolio. Deploys to GitHub Pages.

## Deploy (GitHub Pages)

1. **Turn on GitHub Pages** (one-time):
   - Repo → **Settings** → **Pages**
   - **Source**: **Deploy from a branch**
   - **Branch**: `main` → **Folder**: `/ (root)`
   - Click **Save**

2. **Open the site** (after ~1 minute):  
   **https://makarandpundlik.github.io/new_portfolio/**

Future pushes to `main` will update the site automatically.
